# 🤖 ChatBot - Rule-Based AI Chatbot

A modern desktop **Rule-Based AI ChatBot** developed using **Python** and **Tkinter**. This chatbot responds to user queries using predefined rules and a built-in knowledge base without relying on any AI APIs or Machine Learning models.

This project was developed by **Shiv Pramod Koli** as part of an AI Internship to demonstrate the fundamentals of rule-based conversational systems and GUI application development.

# 🧠 How It Works

The chatbot follows a **Rule-Based Approach**.

1. User enters a message.
2. The message is converted to lowercase.
3. The chatbot checks predefined rules.
4. If a matching keyword is found,
   the chatbot returns the corresponding response.
5. If no rule matches, a default response is displayed.

No Machine Learning or AI APIs are used.
